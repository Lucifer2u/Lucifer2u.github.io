常用工具启动和关闭

mysql

$ mysql.server start
$ mysql.server restart
$ mysql.server stop
$ mysql.server status
$ mysql -u root -p，

nginx

$ service nginx start

$ service nginx stop

$ service nginx reload

Redis

$ redis-server 
$ redis-cli
$ redis-benchmark
#以配置文件启动
$ redis-server /usr/local/redis-2.8.9/redis6379.conf
#以指定端口启动
$ redis-cli -p 6379
#开启哨兵
$ seredis-server sentinel.conf --sentinel

Vim

  • 按esc切换到命令模式，输入半角冒号，输入set空格paste之后看到下面insert有个括号，括号里面有paste再粘贴）
  • dd快速删除一行
  • :set nu显示行号

终端命令

Mac快捷键

//删除当前行 ^+U

打开系统文件

$ open -e .bash_profile
$ source .bash_profile

修改文件权限

$ sudo chmod 777 xx文件名

查看当前进程

$ ps -ef | grep xxx
$ lsof -i:端口号
$ top #动态查看
$ top -H -p pid #查看java线程信息

Docker

#在docker中启动redis，需要用到以下命令
$ docker exec -it (docker容器名) redis-cli -h \
192.168.191.2（ip） -p 6379

Git

$ git init #初始化
#设置项目级别
$ git config user.name lucifer
$ git config user.email xiaoweliang@qq.com
#设置用户级别
$ git config --global user.name lucifer
$ git config --global user.email xiaoweliang@qq.com
#提交
$ git commit -m "commit test" xx.txt
#查看历史版本
$ git relog

#添加远程仓库
$ git remote add origin https://github.com/Lucifer2u/MyFirstTest.gitRedis

Hexo

$ hexo g  # 生成博客网页文件
$ hexo s  # 本地预览博客
$ hexo d  # 上传网页文件到github
$ hexo new post "article title"

JVM

#查看java进程
$ jps
#查看线程快照
$ jstack pid
#综合工具
$ jconsole
